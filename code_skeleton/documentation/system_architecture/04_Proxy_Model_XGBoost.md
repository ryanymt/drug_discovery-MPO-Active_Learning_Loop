# Component: Proxy Model (The Critic)

The Proxy Model serves as a fast surrogate for the expensive Oracle, allowing us to screen millions of candidates in seconds.

## 1. Architecture
*   **Algorithm:** XGBoost Regressor (Gradient Boosted Trees).
*   **Features:** Morgan Fingerprints (Radius 2, 2048 bits) generated via RDKit.
*   **Target:** FEP Binding Free Energy ($\Delta G$).

## 2. Training Workflow
1.  **Ingest:** Query `bioops_platform.validation_results` (BigQuery) to get the latest labeled data (Gold Tier).
2.  **Featurize:** Convert SMILES to Bit Vectors.
3.  **Train:** Fit the XGBoost model.
4.  **Save:** Export `model.json` to GCS.

## 3. Inference Workflow
1.  **Load:** Retrieve the latest `model.json`.
2.  **Predict:** Score the massive "Unlabeled Pool" (e.g., 100,000 molecules generated by Pocket2Mol).
3.  **Rank:** Output a sorted list of candidates.

## 4. Why XGBoost?
*   **Robustness:** Handles small datasets (100-1000 samples) better than Deep Neural Networks.
*   **Speed:** Inference is microsecond-scale.
*   **Interpretability:** Feature importance allows us to sanity-check what chemical features drive affinity.
